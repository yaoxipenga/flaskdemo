欢迎使用本博客系统，这是一个简洁易用的文章发布平台。

🔧 功能概述

  • ✍️ 文章发布：支持在线创建新文章，包括标题与内容填写。
  • 📂 文章列表：首页展示所有已发布文章，按时间排序。
  • 🔍 文章查看：点击标题可查看完整文章内容。
  • 🛠️ 文章编辑：可对已发布文章进行修改。
  • 🗑️ 文章删除：支持删除不需要的文章，带确认提示。
  • 💾 SQLite 数据库存储：数据持久化存储文章信息。
  • 🌐 Flask 框架开发：基于 Python 的 Web 应用，结构清晰。

💡 技术栈

  • 前端：HTML + Bootstrap（美观简洁）
  • 后端：Flask（轻量 Python Web 框架）
  • 数据库：SQLite（零配置、轻量级数据库）

✅ 适用场景

  • 初学者练习 Flask 或 Web 开发
  • 简易个人博客系统
  • 内部文章管理工具原型
